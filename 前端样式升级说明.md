# 前端样式升级说明 - 赛博朋克风格

## 概述

将现有的微信风格界面升级为赛博朋克/暗黑科技风格。

---

## 配色方案

### 主色调
- 背景色：`#0A0A0C` (深黑)
- 次级背景：`#151518` (深灰)
- 卡片背景：`#1E1E24` (灰黑)
- 强调色：`#00E5FF` (霓虹青)
- 警告色：`#FF3366` (霓虹红)

### 文字颜色
- 主文字：`#FFFFFF` / `#E0E0E0`
- 次级文字：`#85858A`
- 占位符：`#555555`

### 边框和分隔线
- 边框：`border-white/5` 或 `border-white/10`
- 强调边框：`border-[#00E5FF]/30` 或 `border-[#00E5FF]/40`

## 特效说明

### 1. 霓虹发光效果

```css
/* 文字发光 */
drop-shadow-[0_0_8px_rgba(0,229,255,1)]

/* 边框发光 */
shadow-[0_0_15px_rgba(0,229,255,0.3)]

/* 按钮悬停发光 */
hover:shadow-[0_0_25px_rgba(0,229,255,0.5)]
```

### 2. 毛玻璃效果

```css
backdrop-blur-xl
bg-[#151518]/90
```

### 3. 渐变背景

```css
/* 径向渐变 */
bg-[radial-gradient(ellipse_at_center,_var(--tw-gradient-stops))] from-[#1E1E24] to-[#0A0A0C]

/* 线性渐变 */
bg-gradient-to-br from-[#00E5FF]/20 to-[#008A99]/5
```

## 注意事项

1. **保持可读性**：虽然是暗黑风格，但文字对比度要足够
2. **性能优化**：发光效果不要过度使用，避免影响性能
3. **一致性**：所有组件使用统一的配色方案
4. **可访问性**：确保按钮和交互元素有足够的点击区域
