## 功能流程

### 创建群聊
1. 点击会话列表头部的 + 按钮
2. 弹出创建群聊弹窗
3. 选择联系人并输入群名
4. 点击"熔铸链接"
5. 调用 API 创建群聊
6. 刷新会话列表并自动选中新群聊

### 查看群成员
1. 在群聊界面点击"潜伏者"按钮
2. 弹出群成员列表
3. 显示群主（首领）和普通成员
4. 显示在线状态

### 群聊消息
1. 群聊中对方消息显示发送者名字（青色荧光）
2. 我方消息不显示名字
3. 私聊消息不显示名字
