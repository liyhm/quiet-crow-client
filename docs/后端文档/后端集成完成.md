## 💡 使用示例

### 完整流程示例

```typescript
import { useAuthStore } from '@/entities/user/model/useAuthStore'
import { useSessionStore } from '@/entities/session/model/useSessionStore'
import { useMessageStore } from '@/entities/message/model/useMessageStore'
import { websocketService } from '@/shared/api/websocket'

// 1. 登录
const authStore = useAuthStore()
const result = await authStore.login({
  username: 'alice',
  password: '123456'
})

if (result.success) {
  // 2. 初始化 WebSocket
  websocketService.initialize()
  websocketService.connect()

  // 3. 加载会话列表
  const sessionStore = useSessionStore()
  await sessionStore.loadSessions()

  // 4. 选择会话
  const sessionId = sessionStore.sessions[0].id
  sessionStore.setActiveSession(sessionId)

  // 5. 加载历史消息
  const messageStore = useMessageStore()
  await messageStore.loadMessages(sessionId)

  // 6. 订阅实时消息
  websocketService.subscribeToChat(sessionId, (message) => {
    messageStore.receiveMessage(message)
  })

  // 7. 发送消息
  await messageStore.sendMessage(sessionId, 'Hello!')
}
```

### 单独使用 API


## 🔐 加密说明

### 加密模式

- 使用 **AES-ECB** 模式（与后端一致）
- 基于 **crypto-js** 库实现
- 会话密钥由后端生成和管理

### 加密流程

```typescript
import { backendCrypto } from '@/shared/lib/cryptoBackend'
import { encryptionApi } from '@/shared/api/encryption'

// 1. 获取会话密钥
const { aesKey } = await encryptionApi.getSessionKey('session-id')

// 2. 设置密钥
backendCrypto.setSessionKey('session-id', aesKey)

// 3. 加密消息
const encrypted = backendCrypto.encryptWithSession('session-id', 'Hello')

// 4. 解密消息
const decrypted = backendCrypto.decryptWithSession('session-id', encrypted)
```

## ⚠️ 注意事项

1. **Token 存储**：Token 存储在 localStorage 的 `chat_token` 键
2. **自动登录**：应用启动时会自动检查 Token 并验证
3. **Token 过期**：Token 过期时会自动跳转到登录页
4. **加密密钥**：会话密钥在内存中管理，刷新页面会重新获取
5. **WebSocket 重连**：目前禁用了自动重连，生产环境需要启用

## 📞 问题排查

### 问题：登录失败

- 检查后端是否运行
- 检查用户名密码是否正确
- 查看 Network 面板的错误信息

### 问题：WebSocket 连接失败

- 检查后端 WebSocket 端点
- 确认 Token 是否有效
- 查看控制台错误信息

### 问题：消息解密失败

- 确认会话密钥是否正确获取
- 检查加密模式是否匹配
- 查看控制台加密日志

### 问题：CORS 错误

- 确保后端配置了 CORS
- 允许来自 `http://localhost:5173` 的请求
