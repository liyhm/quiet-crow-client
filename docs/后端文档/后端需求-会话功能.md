## 后端实现建议

### 1. SessionController.createPrivateSession()

```java
@PostMapping("/private")
public Result<SessionResponse> createPrivateSession(@RequestBody CreatePrivateSessionRequest request) {
    String currentUserId = SecurityUtils.getCurrentUserId();
    String targetUserId = request.getTargetUserId();
    
    // 检查是否已存在会话
    Session existingSession = sessionService.findPrivateSession(currentUserId, targetUserId);
    
    if (existingSession != null) {
        // 会话已存在
        return Result.success(SessionResponse.builder()
            .sessionId(existingSession.getId())
            .type("PRIVATE")
            .participants(Arrays.asList(currentUserId, targetUserId))
            .exists(true)
            .build());
    }
    
    // 创建新会话
    Session newSession = sessionService.createPrivateSession(currentUserId, targetUserId);
    
    return Result.success(SessionResponse.builder()
        .sessionId(newSession.getId())
        .type("PRIVATE")
        .participants(Arrays.asList(currentUserId, targetUserId))
        .exists(false)
        .build());
}
```

### 2. SessionService.createPrivateSession()

```java
@Transactional
public Session createPrivateSession(String userId1, String userId2) {
    // 1. 创建 Session 记录
    Session session = new Session();
    session.setId(UUID.randomUUID().toString());
    session.setSessionType(SessionType.PRIVATE);
    session.setCreatedTime(LocalDateTime.now());
    sessionRepository.save(session);
    
    // 2. 为两个用户创建 SessionUser 记录
    SessionUser sessionUser1 = new SessionUser();
    sessionUser1.setId(UUID.randomUUID().toString());
    sessionUser1.setSessionId(session.getId());
    sessionUser1.setUserId(userId1);
    sessionUser1.setSessionUserType(SessionUserType.PRIVATE);
    sessionUser1.setJoinTime(LocalDateTime.now());
    sessionUserRepository.save(sessionUser1);
    
    SessionUser sessionUser2 = new SessionUser();
    sessionUser2.setId(UUID.randomUUID().toString());
    sessionUser2.setSessionId(session.getId());
    sessionUser2.setUserId(userId2);
    sessionUser2.setSessionUserType(SessionUserType.PRIVATE);
    sessionUser2.setJoinTime(LocalDateTime.now());
    sessionUserRepository.save(sessionUser2);
    
    return session;
}
```

### 3. SessionService.findPrivateSession()

```java
public Session findPrivateSession(String userId1, String userId2) {
    // 查找两个用户共同参与的私聊会话
    List<String> sessionIds1 = sessionUserRepository.findSessionIdsByUserId(userId1);
    List<String> sessionIds2 = sessionUserRepository.findSessionIdsByUserId(userId2);
    
    // 找到共同的会话ID
    sessionIds1.retainAll(sessionIds2);
    
    if (sessionIds1.isEmpty()) {
        return null;
    }
    
    // 返回第一个私聊会话
    for (String sessionId : sessionIds1) {
        Session session = sessionRepository.findById(sessionId);
        if (session.getSessionType() == SessionType.PRIVATE) {
            return session;
        }
    }
    
    return null;
}
```

## 数据库表结构

### session 表
```sql
CREATE TABLE session (
    id VARCHAR(36) PRIMARY KEY,
    session_type VARCHAR(20) NOT NULL,  -- PRIVATE, GROUP
    created_time TIMESTAMP NOT NULL,
    updated_time TIMESTAMP
);
```

### session_user 表
```sql
CREATE TABLE session_user (
    id VARCHAR(36) PRIMARY KEY,
    session_id VARCHAR(36) NOT NULL,
    user_id VARCHAR(36) NOT NULL,
    session_user_type VARCHAR(20) NOT NULL,  -- PRIVATE, GROUP
    join_time TIMESTAMP NOT NULL,
    FOREIGN KEY (session_id) REFERENCES session(id),
    FOREIGN KEY (user_id) REFERENCES user(id)
);
```

### message 表
```sql
CREATE TABLE message (
    id VARCHAR(36) PRIMARY KEY,
    session_id VARCHAR(36) NOT NULL,
    send_user VARCHAR(36) NOT NULL,
    message_content TEXT NOT NULL,
    message_type VARCHAR(20) NOT NULL,  -- TEXT, IMAGE, FILE
    send_time TIMESTAMP NOT NULL,
    is_read BOOLEAN DEFAULT FALSE,
    FOREIGN KEY (session_id) REFERENCES session(id),
    FOREIGN KEY (send_user) REFERENCES user(id)
);
```
