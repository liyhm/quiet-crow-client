## ✅ 解决方案

### 1. 私聊会话返回头像

**需要修改的接口**：`GET /api/session/my`

**逻辑**：
1. 查询会话列表时，关联查询用户表
2. 获取对方用户的头像字段
3. 如果头像存在，返回头像 URL
4. 如果头像不存在，返回 `null`（前端会显示首字母）

**示例代码**：
```java
@Service
public class SessionService {
    
    @Autowired
    private UserRepository userRepository;
    
    public List<SessionUserDto> getMySessions(String currentUserId) {
        // 1. 查询当前用户的所有会话
        List<SessionUser> sessionUsers = sessionUserRepository
            .findByUserIdAndDeletedAtIsNull(currentUserId);
        
        // 2. 转换为 DTO 并填充头像
        return sessionUsers.stream()
            .map(su -> {
                SessionUserDto dto = new SessionUserDto();
                dto.setId(su.getId());
                dto.setSessionId(su.getSessionId());
                dto.setUserId(su.getUserId());
                dto.setSessionUserType(su.getSessionUserType());
                dto.setLastMessage(su.getLastMessage());
                dto.setLastMessageTime(su.getLastMessageTime());
                dto.setUnreadCount(su.getUnreadCount());
                dto.setIsOnline(onlineStatusService.isUserOnline(su.getUserId()));
                
                // 3. 根据会话类型填充头像
                if ("PRIVATE".equals(su.getSessionUserType())) {
                    // 私聊：查询对方用户的头像
                    String otherUserId = getOtherUserId(su.getSessionId(), currentUserId);
                    User otherUser = userRepository.findById(otherUserId).orElse(null);
                    
                    if (otherUser != null) {
                        dto.setShowName(otherUser.getShowName());
                        // 设置头像 URL
                        if (otherUser.getAvatar() != null && !otherUser.getAvatar().isEmpty()) {
                            dto.setAvatar("/api/user/avatar/" + otherUserId);
                        } else {
                            dto.setAvatar(null);
                        }
                    }
                } else if ("GROUP".equals(su.getSessionUserType())) {
                    // 群聊：查询群聊信息
                    Session session = sessionRepository.findById(su.getSessionId()).orElse(null);
                    if (session != null) {
                        dto.setShowName(session.getName());
                        // 群聊可以设置群头像（可选）
                        if (session.getAvatar() != null && !session.getAvatar().isEmpty()) {
                            dto.setAvatar("/api/session/avatar/" + su.getSessionId());
                        } else {
                            dto.setAvatar(null);
                        }
                        dto.setMemberCount(session.getMemberCount());
                    }
                }
                
                return dto;
            })
            .collect(Collectors.toList());
    }
    
    /**
     * 获取私聊会话中的对方用户 ID
     */
    private String getOtherUserId(String sessionId, String currentUserId) {
        List<SessionUser> sessionUsers = sessionUserRepository
            .findBySessionIdAndDeletedAtIsNull(sessionId);
        
        return sessionUsers.stream()
            .map(SessionUser::getUserId)
            .filter(userId -> !userId.equals(currentUserId))
            .findFirst()
            .orElse(null);
    }
}
```

### 2. 头像 URL 格式

**推荐格式**：
- 私聊头像：`/api/user/avatar/{userId}`
- 群聊头像：`/api/session/avatar/{sessionId}` （可选）

**前端会自动拼接完整 URL**：
- 开发环境：`http://localhost:8080/api/user/avatar/{userId}`
- 生产环境：`https://your-domain.com/api/user/avatar/{userId}`

### 3. 头像接口

确保头像接口已实现：

```java
@RestController
@RequestMapping("/api/user")
public class UserController {
    
    @Autowired
    private UserService userService;
    
    /**
     * 获取用户头像
     */
    @GetMapping("/avatar/{userId}")
    public ResponseEntity<byte[]> getUserAvatar(@PathVariable String userId) {
        try {
            byte[] avatarData = userService.getUserAvatar(userId);
            
            if (avatarData == null || avatarData.length == 0) {
                return ResponseEntity.notFound().build();
            }
            
            HttpHeaders headers = new HttpHeaders();
            headers.setContentType(MediaType.IMAGE_JPEG);
            headers.setCacheControl(CacheControl.maxAge(7, TimeUnit.DAYS));
            
            return ResponseEntity.ok()
                .headers(headers)
                .body(avatarData);
        } catch (Exception e) {
            log.error("获取用户头像失败: userId={}", userId, e);
            return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).build();
        }
    }
}
```

## ⚠️ 注意事项

### 1. 性能优化

**问题**：查询会话列表时，需要关联查询用户表，可能影响性能

**优化方案**：
1. 使用 JOIN 查询，一次性获取所有数据
2. 添加数据库索引：`user_id`、`session_id`
3. 考虑缓存用户头像 URL（Redis）

### 2. 头像 URL 格式

**推荐**：返回相对路径
```json
{
  "avatar": "/api/user/avatar/4fda11c3d3ff5cee46e7380b7b193e56"
}
```

**不推荐**：返回完整 URL
```json
{
  "avatar": "http://localhost:8080/api/user/avatar/4fda11c3d3ff5cee46e7380b7b193e56"
}
```

**原因**：
- 前端会自动拼接 BASE_URL
- 支持开发/生产环境切换
- 避免硬编码域名

### 3. 群聊头像

群聊头像是可选的：
- 如果群聊有设置头像，返回头像 URL
- 如果没有设置，返回 `null`，前端显示群聊图标

### 4. 头像缓存

建议添加缓存控制：
```java
headers.setCacheControl(CacheControl.maxAge(7, TimeUnit.DAYS));
```

**好处**：
- 减少服务器压力
- 提高前端加载速度
- 用户体验更好
