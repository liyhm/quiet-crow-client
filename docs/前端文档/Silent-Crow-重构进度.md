## 🔧 快速替换指南

### 全局替换（使用 VS Code 正则）

1. **背景色**
   - 查找：`bg-workspace-bg|bg-list-gray`
   - 替换：`bg-bg-void` 或 `bg-bg-surface`（根据上下文）

2. **边框**
   - 查找：`border-gray-300|border-gray-200`
   - 替换：`border-white/10`

3. **主文字**
   - 查找：`text-wechat-text|text-gray-700`
   - 替换：`text-text-primary`

4. **次要文字**
   - 查找：`text-gray-400|text-gray-500|text-gray-600`
   - 替换：`text-text-muted`

5. **按钮背景**
   - 查找：`bg-gray-100|bg-gray-200`
   - 替换：`bg-white/5`

6. **Hover 效果**
   - 查找：`hover:bg-gray-200|hover:bg-gray-300`
   - 替换：`hover:bg-white/5` 或 `hover:bg-white/10`

7. **微信绿色**
   - 查找：`style="background-color: #95EC69"`
   - 删除，使用新的霓虹青色样式

8. **红色警告**
   - 查找：`bg-red-500`
   - 替换：`bg-alert-red`
