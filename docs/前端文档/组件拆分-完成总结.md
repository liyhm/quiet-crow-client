## 🔧 技术细节

### 组件通信方式
- **Props Down**: 父组件向子组件传递数据
- **Events Up**: 子组件通过事件向父组件通信
- **v-model**: 双向绑定（MessageInput, ProfileModal）
- **Ref**: 父组件调用子组件方法（MessageList.scrollToBottom）

### 类型安全
- 所有组件使用 TypeScript
- Props 和 Events 都有明确的类型定义
- 使用 `defineProps<T>()` 和 `defineEmits<T>()` 确保类型安全

### 样式一致性
- 所有组件使用统一的 Tailwind 配色
- 使用 Silent Crow 赛博朋克暗黑主题
- 保持毛玻璃效果和霓虹发光效果

      @delete="handleDelete"

      v-model="message"
      @send="handleSend"
    />
  </div>

import { ref } from 'vue'

const message = ref('')

const handleDelete = () => {
  console.log('删除消息')
}

const handleSend = () => {
  console.log('发送:', message.value)
  message.value = ''
}
</script>
