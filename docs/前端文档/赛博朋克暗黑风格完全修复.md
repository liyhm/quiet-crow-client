## 修复概述

完成了主界面的高级 UI/UX 重构，实现了完整的赛博朋克暗黑风格设计系统。

## 修复内容

### 1. 全局样式增强 (`global.css`)

添加了跑马灯动画和渐变遮罩效果：

```css
/* 跑马灯无限滚动动画 */
@keyframes marquee {
  0% { transform: translateX(0); }
  100% { transform: translateX(-50%); }
}

.animate-marquee {
  width: max-content;
  animation: marquee 20s linear infinite;
}

/* 标题栏左右两端渐变消失，营造融入感 */
.mask-fade-edges {
  mask-image: linear-gradient(to right, transparent, black 5%, black 95%, transparent);
  -webkit-mask-image: linear-gradient(to right, transparent, black 5%, black 95%, transparent);
}
```

**在线状态**：
- 青色滚动文字：`SYS.ONLINE 🦅 正在监听鸦群...`

**离线状态**：

- 方形闪烁指示器

### 3. 左侧导航栏头像升级 (`Sidebar.vue`)

**头像判断逻辑**：

**交互效果**：
- 青色边框发光：`border-[#00E5FF]/30`
- Hover 时增强发光：`hover:shadow-[0_0_20px_rgba(0,229,255,0.3)]`
- 图片/Logo 缩放动画：`group-hover:scale-110`

### 4. 会话列表搜索框升级 (`SessionList.vue`)

### 5. 会话列表卡片化设计 (`SessionList.vue`)

**未选中状态**：
- 深色毛玻璃卡片：`bg-[#151518]/50 backdrop-blur-sm`

- 淡阴影：`shadow-[0_2px_8px_rgba(0,0,0,0.3)]`
- Hover 时边框变亮：`hover:border-white/10`

**选中状态**：
- 青色左侧高亮条：`border-l-4 border-[#00E5FF]`
- 青色辉光背景：`bg-[#00E5FF]/5 blur-md`（绝对定位）
- 整体发光：`shadow-[0_0_20px_rgba(0,229,255,0.15)]`
- 文字变青色：`text-[#00E5FF]`

**头像样式**：
- 未选中：深色背景 + 暗灰边框
- 选中：黑色背景 + 青色边框 + 发光阴影

## 设计理念

### 终端式跑马灯（Marquee）
像黑客电影里的监控终端，顶部持续滚动系统状态。在线时显示青色监听状态，断网时立刻变成红色高危报警，氛围感拉满。

### 头像降级处理
如果用户上传了真实头像，完美展示；如果没有，显示带呼吸灯的赛博乌鸦 Logo，不再是干巴巴的字母。

### 搜索框内凹 + 图标激活
深色内阴影营造凹陷感，左侧放大镜图标平时暗灰，聚焦时和边框同时亮起青色霓虹光。

### 会话列表卡片化（Card UI）
- 未选中：微透的深灰卡片，带淡淡外阴影
- 选中：左侧青色能量条 + 整体青色辉光渐变
- 头像使用圆角方形（`rounded-lg`），更符合赛博朋克几何美学

## 技术亮点

1. **纯 CSS 动画**：跑马灯使用 `@keyframes` 实现无缝循环，无需 JavaScript
2. **渐变遮罩**：使用 `mask-image` 让滚动文字自然融入边缘
3. **条件渲染**：根据连接状态和选中状态动态切换样式
4. **分层设计**：使用 `relative z-10` 确保内容在辉光背景之上
5. **组合选择器**：`group-focus-within` 实现父级聚焦时子元素变色
