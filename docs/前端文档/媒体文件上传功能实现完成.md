## 🔐 安全性

### 加密机制
1. **文件内容加密**: 使用会话密钥（AES-256-GCM）加密文件内容
2. **文件名加密**: 文件名也使用会话密钥加密，防止泄露文件信息
3. **端到端加密**: 文件在客户端加密后上传，服务器无法解密
4. **随机 IV**: 每次加密使用随机生成的 IV，确保安全性

### 验证机制
1. **文件类型验证**: 仅允许图片和视频文件
2. **文件大小验证**: 图片 5MB，视频 50MB
3. **会话权限验证**: 必须是会话成员才能上传文件

## 💡 使用示例

### 前端调用示例

```typescript
// 1. 选择文件
const input = document.createElement('input')
input.type = 'file'
input.accept = 'image/*,video/*'
input.onchange = async (e) => {
  const files = (e.target as HTMLInputElement).files
  if (files) {
    await handleFiles(Array.from(files))
  }
}
input.click()

// 2. 上传文件
const uploadFile = async (file: File) => {
  // 获取会话密钥
  const sessionKey = await getSessionKey(sessionId)
  
  // 加密文件
  const { encryptedData } = await FileCryptoService.encryptFile(file, sessionKey)
  
  // 加密文件名
  const encryptedFileName = await FileCryptoService.encryptFileName(file.name, sessionKey)
  
  // 上传
  const result = await mediaApi.uploadMedia({
    file: new Blob([encryptedData]),
    metadata: {
      sessionId,
      fileName: encryptedFileName,
      fileSize: file.size,
      mimeType: file.type,
      messageType: file.type.startsWith('image/') ? 'IMAGE' : 'VIDEO'
    }
  }, (progress) => {
    console.log('上传进度:', progress + '%')
  })
  
  // 发送消息
  const messageContent = JSON.stringify({
    fileId: result.fileId,
    fileName: encryptedFileName,
    fileSize: file.size
  })
  
  websocketService.sendMessage(sessionId, {
    sessionId,
    content: encryptWithSession(sessionId, messageContent),
    encrypted: true,
    messageType: file.type.startsWith('image/') ? '2' : '3'
  })
}
```
