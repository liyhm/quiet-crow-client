## 如何使用新组件？

### 方案 1: 渐进式迁移（推荐）

逐个替换 MainPage.vue 中的部分，而不是一次性全部替换：

#### 步骤 1: 先替换简单组件

```vue
<!-- 在 MainPage.vue 中 -->
<script setup lang="ts">
// 添加导入
import EmptyState from '@/widgets/emptyState/EmptyState.vue'
import TitleBar from '@/widgets/titleBar/TitleBar.vue'
</script>

<template>
  <!-- 替换标题栏 -->
  <TitleBar
    :title="`鸦口无言 - ${connectionStatusText}`"
    @minimize="minimizeWindow"
    @maximize="maximizeWindow"
    @close="closeWindow"
  />
  
  <!-- 替换空状态 -->
  <EmptyState v-if="!activeSession" />
</template>
```

#### 步骤 2: 测试后再替换复杂组件

确认简单组件工作正常后，再替换 MessageBubble、MessageList 等。

### 方案 2: 创建新页面测试

创建一个新的测试页面来使用这些组件：

```vue
<!-- src/renderer/src/pages/test/TestPage.vue -->
<template>
  <div class="h-full w-full flex flex-col bg-bg-void">
    <TitleBar
      title="测试页面"
      @minimize="minimizeWindow"
      @maximize="maximizeWindow"
      @close="closeWindow"
    />
    
    <MessageBubble
      content="测试消息"
      timestamp="2024-02-27T10:00:00Z"
      :is-mine="true"
      sender-name="测试用户"
    />
  </div>
</template>

<script setup lang="ts">
import TitleBar from '@/widgets/titleBar/TitleBar.vue'
import MessageBubble from '@/widgets/messageBubble/MessageBubble.vue'

const minimizeWindow = async () => {
  await window.api.window.minimize()
}

const maximizeWindow = async () => {
  await window.api.window.maximize()
}

const closeWindow = async () => {
  await window.api.window.close()
}
</script>
```

}
