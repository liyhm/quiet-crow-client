## 🔑 重要配置文件说明

### electron-builder.yml
Electron 打包配置，包含：
- Windows/Mac/Linux 打包设置
- 自动更新服务器配置
- 应用图标和资源配置

### package.json
项目配置文件，包含：
- 项目名称和版本号
- 依赖包列表
- 构建和开发脚本

### .env.development / .env.production
环境变量配置：
- API 服务器地址
- WebSocket 服务器地址
- 其他环境相关配置

### electron.vite.config.ts
Vite 构建配置：
- 主进程、渲染进程、预加载脚本的构建配置
- 开发服务器配置
- 插件配置

## 🚀 常用命令

### 开发
```bash
npm run dev          # 启动开发服务器
npm run typecheck    # 类型检查
npm run lint         # 代码检查
npm run format       # 代码格式化
```

### 构建
```bash
npm run build        # 构建应用
npm run build:win    # 构建 Windows 版本
npm run build:mac    # 构建 Mac 版本
npm run build:linux  # 构建 Linux 版本
```
