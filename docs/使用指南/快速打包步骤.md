# 快速打包步骤

## 一键打包（推荐）

### Windows 用户

```bash
npm run build:win
```

等待 3-5 分钟，打包完成后：
- 安装包位置：`dist/quiet-1.0.0-setup.exe`
- 文件大小：约 100-150 MB

### macOS 用户

```bash
npm run build:mac
```

等待 3-5 分钟，打包完成后：
- 安装包位置：`dist/quiet-1.0.0.dmg`

### Linux 用户

```bash
npm run build:linux
```

等待 3-5 分钟，打包完成后：
- AppImage：`dist/quiet-1.0.0.AppImage`
- Deb 包：`dist/quiet-1.0.0.deb`

## 常见错误和解决方案

### 错误 1: `npm ERR! code ELIFECYCLE`

**原因**: 编译失败

**解决**:
```bash
# 清理缓存
npm cache clean --force

# 删除 node_modules
Remove-Item -Recurse -Force node_modules

# 重新安装
npm install

# 重新打包
npm run build:win
```

### 错误 2: `Error: Cannot find module`

**原因**: 依赖缺失

**解决**:
```bash
npm install
```

### 错误 3: `ENOENT: no such file or directory`

**原因**: 构建目录不存在

**解决**:
```bash
# 先编译代码
npm run build

# 再打包
electron-builder --win
```

### 错误 4: 网络超时

**原因**: 下载 Electron 二进制文件失败

**解决**: 已配置国内镜像，如果还是失败，可以手动设置代理：
```bash
# 设置 npm 代理
npm config set proxy http://your-proxy:port
npm config set https-proxy http://your-proxy:port

# 或使用淘宝镜像
npm config set registry https://registry.npmmirror.com
```

### 错误 5: 打包后无法连接后端

**原因**: 生产环境配置错误

**解决**: 检查 `.env.production` 文件，确保地址正确：
```env
VITE_API_BASE_URL=http://159.75.182.85:8081
VITE_WS_URL=http://159.75.182.85:8081/ws
```

## 性能优化建议

### 1. 减小打包体积

在 `electron-builder.yml` 中添加：
```yaml
compression: maximum
```

### 2. 排除不必要的文件

在 `electron-builder.yml` 中添加：
```yaml
files:
  - '!**/.git/*'
  - '!**/node_modules/*/{CHANGELOG.md,README.md,README,readme.md,readme}'
  - '!**/node_modules/*/{test,__tests__,tests,powered-test,example,examples}'
  - '!**/node_modules/*.d.ts'
  - '!**/node_modules/.bin'
```

### 3. 使用 asar 压缩

默认已启用，如需禁用：
```yaml
asar: false
```
