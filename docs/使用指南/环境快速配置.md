# 环境快速配置

## 开发环境

1. 复制环境配置文件：
```bash
copy .env.example .env.development
```

2. 编辑 `.env.development`：
```env
VITE_API_BASE_URL=http://localhost:8080
VITE_WS_URL=ws://localhost:8080/ws
```

3. 启动后端（端口 8080）

4. 启动前端：
```bash
npm run dev
```

## 生产环境

1. 编辑 `.env.production`：
```env
VITE_API_BASE_URL=http://159.75.182.85:8080
VITE_WS_URL=ws://159.75.182.85:8080/ws
```

2. 打包：
```bash
npm run build
```

## 注册和登录

- 首次使用需要先注册账号
- 点击"没有账号？去注册"切换到注册页面
- 注册成功后会自动登录并生成加密密钥
