# 联系人功能测试说明

## 当前问题分析

根据代码审查，发现以下问题：

### 1. 联系人列表加载问题

**问题**: `useContactStore.ts` 中的 `loadContacts()` 方法错误地为每个联系人调用 `userApi.getInfo()`，这会返回当前用户的信息，而不是联系人的信息。

**原因**: 
```typescript
// ❌ 错误的实现
const userInfo = await userApi.getInfo()  // 这返回的是当前用户信息！
return {
  userId: relation.toUserId,
  username: userInfo.username,  // 错误：这是当前用户的用户名
  displayName: userInfo.showName,  // 错误：这是当前用户的昵称
  ...
}
```

**修复**: 
根据后端文档，`GET /api/contact/list` 应该直接返回包含用户信息的联系人列表：

```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "id": "relation-uuid-123",
      "userId": "user-uuid-456",
      "username": "lisi",
      "showName": "李四",
      "avatar": "http://...",
      "remark": "同事",
      "addedTime": "2024-01-01T10:00:00",
      "publicKey": "-----BEGIN PUBLIC KEY-----..."
    }
  ]
}
```

因此，我们应该直接使用后端返回的数据，而不是再次调用 API。

### 2. ContactRelation 接口定义不匹配

**问题**: 前端定义的 `ContactRelation` 接口与后端返回的数据结构不一致。

**修复前**:
```typescript
export interface ContactRelation {
  id: string
  fromUserId: string
  toUserId: string
  status: 'PENDING' | 'ACCEPTED' | 'REJECTED'
  remark: string | null
}
```

**修复后**:
```typescript
export interface ContactRelation {
  id: string
  userId: string
  username: string
  showName: string
  avatar: string
  remark: string | null
  addedTime: string
  publicKey: string
  status?: 'PENDING' | 'ACCEPTED' | 'REJECTED'
}
```

### 2. 修复 loadContacts 方法
✅ 已修复 `src/renderer/src/entities/contact/model/useContactStore.ts` 中的方法实现

新的实现：
```typescript
const loadContacts = async () => {
  try {
    loading.value = true
    console.log('📇 加载联系人列表...')
    
    const relations = await contactsApi.getList()
    console.log('✅ 联系人关系:', relations)

    // 后端返回的联系人列表已经包含了用户信息
    // 直接映射到 Contact 接口
    contacts.value = relations
      .filter((r) => r.status === 'ACCEPTED')
      .map((relation: any) => ({
        userId: relation.userId || relation.toUserId,
        username: relation.username || 'unknown',
        displayName: relation.showName || relation.username || '未知用户',
        avatar: relation.avatar || '',
        status: 'ONLINE',
        statusMessage: ''
      }))
    
    console.log('✅ 联系人列表加载完成:', contacts.value.length)
    return { success: true }
  } catch (error) {
    console.error('❌ 加载联系人失败:', error)
    return { success: false, error: String(error) }
  } finally {
    loading.value = false
  }
}
```

## 可能的错误

### 1. 后端返回 405 Method Not Allowed
**原因**: HTTP 方法不匹配
**解决**: 确认后端是否同时支持 POST 和 PUT 方法

### 2. 联系人列表为空
**原因**: 
- 好友关系未正确建立
- 后端返回的数据格式不匹配
**解决**: 
- 检查后端日志
- 检查数据库中的 contact_relation 表

### 3. 显示用户ID而不是用户名
**原因**: 后端未返回 username 和 showName 字段
**解决**: 
- 检查后端 API 实现
- 确认后端是否正确关联用户表
