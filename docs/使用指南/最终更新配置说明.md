## ⚠️ 重要提示

### 1. 文件名必须精确匹配
- Windows：必须是 `quiet-latest.yml`
- Mac：必须是 `quiet-latest-mac.yml`
- 不能是其他名字

### 2. Mac 必须有 ZIP 文件
- 必须上传 `quiet-{version}-mac.zip`
- 不能只有 `.dmg` 文件
- `.dmg` 用于首次安装，`.zip` 用于自动更新

### 3. 版本号格式
- 必须是 `x.y.z` 格式（如 `1.0.0`）
- 不能有 `v` 前缀（如 `v1.0.0`）
- 必须是三位数字

### 4. SHA512 必须正确
- 如果手动修改 yml 文件，必须重新计算 SHA512
- 如果 SHA512 不匹配，更新会失败
- 建议使用自动生成的 yml 文件
