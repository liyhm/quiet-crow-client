# 自动更新快速参考

## 🎯 一句话总结

Windows 和 Mac 应用都会自动检查 `http://159.75.182.85/updates/` 的更新，Windows 读取 `latest.yml`，Mac 读取 `latest-mac.yml`。

---

## 📦 服务器必需文件

```
http://159.75.182.85/updates/
├── latest.yml              ← Windows 版本信息（必需）
├── latest-mac.yml          ← Mac 版本信息（必需）
├── quiet-1.0.0-setup.exe   ← Windows 安装包
└── quiet-1.0.0-mac.zip     ← Mac 更新包（必需，不是 .dmg）
```

---

## 🚀 发布新版本（3 步）

### 1. 修改版本号
```json
// package.json
{
  "version": "1.0.1"
}
```

### 2. 推送触发构建
```bash
git add package.json
git commit -m "chore: bump version to 1.0.1"
git push github master
```

### 3. 下载并上传
```bash
# 等待 10-15 分钟构建完成
# 从 GitHub Actions 下载 quiet-windows 和 quiet-macos
# 上传到服务器
scp quiet-windows/* root@159.75.182.85:/var/www/updates/
scp quiet-macos/* root@159.75.182.85:/var/www/updates/
```

---

## ⚠️ 关键注意事项

| 平台 | 必需文件 | 格式 |
|------|---------|------|
| Windows | `latest.yml` + `.exe` | EXE 安装包 |
| Mac | `latest-mac.yml` + `.zip` | ZIP 压缩包（不是 DMG） |

**Mac 特别注意：**
- ❌ 只有 `.dmg` 不能自动更新
- ✅ 必须有 `.zip` 文件
- ✅ `.dmg` 用于首次安装，`.zip` 用于自动更新

---

## 🧪 测试更新

1. 安装旧版本（如 1.0.0）
2. 上传新版本文件到服务器（如 1.0.1）
3. 打开应用，等待 5 秒
4. 应该弹出更新提示

## 🔧 故障排查

### 没有检测到更新？

```bash
# 检查文件是否存在
curl http://159.75.182.85/updates/latest.yml
curl http://159.75.182.85/updates/latest-mac.yml

# 检查版本号
curl http://159.75.182.85/updates/latest.yml | grep version
```

### Mac 更新失败？

```bash
# 确保有 ZIP 文件
curl -I http://159.75.182.85/updates/quiet-1.0.1-mac.zip

# 检查 electron-builder.yml
mac:
  target:
    - dmg
    - zip  # 必须有这一行
```
