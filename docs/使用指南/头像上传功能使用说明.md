# 头像上传功能使用说明

## ✅ 已完成的功能

前端已实现完整的头像上传和个人信息编辑功能。

## 📍 功能位置

在主页面（MainPage）中：

1. 点击左侧边栏顶部的用户头像
2. 或点击左侧边栏底部的"设置"按钮 → 选择"个人信息"

## 🎨 功能特性

### 1. 头像上传

- **支持格式**: jpg, jpeg, png, gif
- **文件大小限制**: 最大 5MB
- **上传方式**: 
  - 点击头像上的相机图标
  - 或点击"更换头像"按钮
- **实时预览**: 上传成功后立即显示新头像
- **自动更新**: 头像会自动同步到所有显示位置

### 2. 个人信息编辑

可编辑字段：
- **昵称**: 必填，显示在聊天界面
- **手机号**: 选填，用于联系方式

只读字段：
- **用户名**: 不可修改
- **用户 ID**: 系统生成的唯一标识

### 3. 用户体验优化

- ✅ 文件类型验证
- ✅ 文件大小验证
- ✅ 上传进度提示
- ✅ 错误提示
- ✅ 成功提示
- ✅ 防止重复提交

## 🔧 技术实现

### API 接口

#### 1. 上传头像
```
POST /api/user/avatar
Content-Type: multipart/form-data
Authorization: Bearer {token}

Body: file (图片文件)
```

#### 2. 更新个人信息
```
PUT /api/user/me
Content-Type: application/json
Authorization: Bearer {token}

Body: {
  "displayName": "昵称",
  "phone": "手机号"
}
```

#### 3. 获取用户信息
```
GET /api/user/info
Authorization: Bearer {token}
```

## 📝 使用流程

### 更换头像

1. 打开个人信息弹窗
2. 点击头像或"更换头像"按钮
3. 选择图片文件（jpg/png/gif，最大5MB）
4. 等待上传完成
5. 查看新头像

### 编辑个人信息

1. 打开个人信息弹窗
2. 修改昵称或手机号
3. 点击"保存修改"按钮
4. 等待保存完成
5. 查看更新后的信息

## ⚠️ 注意事项

### 1. 文件限制

- 只支持图片格式（jpg, jpeg, png, gif）
- 文件大小不能超过 5MB
- 建议使用正方形图片，尺寸 500x500 像素

### 2. 网络要求

- 需要稳定的网络连接
- 上传大文件可能需要较长时间
- 上传失败会有错误提示

### 3. 权限要求

- 需要登录状态（有效的 JWT Token）
- Token 过期会自动跳转到登录页

### 4. 数据同步

- 头像上传成功后会自动刷新用户信息
- 个人信息保存后会自动更新显示
- 所有位置的头像会同步更新

## 🐛 常见问题

### Q1: 上传失败怎么办？

**可能原因**:
- 文件格式不支持
- 文件大小超过限制
- 网络连接问题
- Token 过期

**解决方法**:
1. 检查文件格式和大小
2. 检查网络连接
3. 重新登录获取新 Token
4. 查看浏览器控制台错误信息

### Q2: 头像上传成功但不显示？

**解决方法**:
1. 刷新页面
2. 检查后端返回的头像 URL 是否正确
3. 检查图片服务器是否可访问
4. 查看浏览器控制台网络请求

### Q3: 昵称修改不生效？

**解决方法**:
1. 确保昵称不为空
2. 检查是否点击了"保存修改"按钮
3. 查看是否有错误提示
4. 刷新页面重新查看

### Q4: 如何压缩图片？

**建议**:
- 使用在线图片压缩工具
- 使用图片编辑软件调整尺寸
- 推荐尺寸: 500x500 像素
- 推荐格式: jpg（文件较小）
