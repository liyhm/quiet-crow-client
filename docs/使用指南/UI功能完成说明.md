## 使用说明

### 注册和登录
1. 首次使用点击"没有账号？去注册"
2. 填写用户名、昵称、密码
3. 注册成功后自动登录

### 添加好友
1. 点击联系人标签
2. 点击右上角 "+" 按钮
3. 输入对方的用户 ID
4. 填写验证消息
5. 发送好友请求

### 开始聊天
1. 在联系人列表点击好友
2. 自动创建会话并切换到消息标签
3. 在输入框输入消息
4. 按 Enter 发送（Shift+Enter 换行）

### 查看个人信息
1. 点击左侧边栏顶部的头像
2. 查看个人信息
3. 可以退出登录

## 后端 API 对接状态

### 已对接 ✅
- 用户注册 `/api/auth/register`
- 用户登录 `/api/auth/login`
- 获取用户信息 `/api/users/me`
- 获取会话列表 `/api/session/my`
- 创建私聊会话 `/api/session/private`
- 获取联系人列表 `/api/contacts`
- 发送好友请求 `/api/contacts/requests`
- 接受/拒绝好友请求 `/api/contacts/requests/{id}/accept|reject`

## 注意事项

1. **清除旧数据**：如果遇到 403 错误，在浏览器控制台执行：
   ```javascript
   localStorage.clear()
   location.reload()
   ```

2. **后端要求**：
   - 后端必须启动在 `http://localhost:8080`
   - CORS 已配置
   - 数据库已初始化

3. **环境配置**：
   - 开发环境：`.env.development`
   - 生产环境：`.env.production`
