# Quiet Crow - 快速开始指南

## 🚀 启动应用

### 1. 启动开发服务器

```bash
npm run dev
```

应用将自动启动，显示登录页面。

### 2. 登录测试

- **用户名**: 任意输入（例如：test）
- **密码**: 任意输入（例如：123456）

点击"登录"按钮后：

1. 系统自动生成 RSA 密钥对
2. 私钥通过 OS 级 safeStorage 加密存储
3. 初始化 WebSocket 连接
4. 跳转到主界面

### 3. 主界面功能

#### 左侧栏（60px）

- 用户头像（显示用户名首字母）
- 底部菜单按钮

#### 中间会话列表（250px）

- 搜索框：搜索联系人或聊天记录
- 会话列表：
  - 显示头像、名称、最新消息
  - 未读消息红点提示
  - 点击切换聊天

#### 右侧工作区

- **标题栏**：
  - 显示当前聊天对象
  - 窗口控制按钮（最小化、最大化、关闭）
  - 可拖拽移动窗口
- **消息区域**：
  - 对方消息：左侧白色气泡
  - 自己消息：右侧绿色气泡
  - 支持文件拖拽（UI 已实现）
- **输入区域**：
  - 工具栏：表情、文件上传按钮
  - 输入框：
    - `Enter` 发送消息
    - `Shift + Enter` 换行

### 4. 测试消息加密

当前使用 Mock 数据，消息发送流程：

1. 输入消息内容
2. 按 `Enter` 发送
3. 系统自动：
   - 使用 AES-GCM 加密消息
   - 通过 WebSocket 发送
   - 更新消息状态（发送中 → 已发送）

## 🔐 加密机制

### RSA 密钥对

- **算法**: RSA-OAEP 2048-bit
- **用途**: 加密 AES 会话密钥
- **存储**: OS 级安全存储（Windows DPAPI / macOS Keychain）

### AES 会话密钥

- **算法**: AES-GCM 256-bit
- **用途**: 加密实际消息内容
- **IV**: 12 字节随机生成
- **认证标签**: 128-bit

### 消息加密流程

```
明文消息 → AES-GCM 加密 → 密文 + IV + AuthTag → WebSocket 发送
```

### 消息解密流程

```
接收消息 → 提取密文 + IV + AuthTag → AES-GCM 解密 → 明文消息
```

## 🌐 WebSocket 连接

### 服务器地址

```
ws://159.75.182.85:8080/ws
```

### 连接特性

- **协议**: STOMP over SockJS
- **心跳**: 10 秒间隔
- **重连**: 指数退避算法
  - 延迟公式: `min(1000 * 2^attempts, 30000)ms`
  - 最大延迟: 30 秒

### 订阅频道

- **私人消息**: `/user/queue/messages`
- **会话消息**: `/topic/sessions/{sessionId}`

### 发送消息

- **目标**: `/app/chat/{sessionId}`
- **同步**: `/app/sync`

## 📁 项目结构

```
src/
├── main/                    # Electron 主进程
│   └── index.ts            # 窗口管理、IPC 处理
├── preload/                # 预加载脚本
│   ├── index.ts           # API 暴露
│   └── index.d.ts         # 类型定义
└── renderer/              # Vue 渲染进程
    └── src/
        ├── app/           # 应用配置
        │   ├── router/    # 路由
        │   ├── providers/ # Pinia
        │   └── styles/    # 全局样式
        ├── pages/         # 页面组件
        │   ├── login/     # 登录页
        │   └── main/      # 主页面
        ├── entities/      # 业务实体
        │   ├── user/      # 用户/认证
        │   ├── session/   # 会话
        │   ├── message/   # 消息
        │   ├── crypto/    # 加密
        │   └── network/   # 网络
        └── shared/        # 共享资源
            ├── api/       # WebSocket
            ├── lib/       # 加密工具
            ├── types/     # 类型定义
            └── ui/        # UI 组件
```

## 🐛 调试技巧

### 查看加密密钥

```javascript
// 在 DevTools Console 中
const cryptoStore = useCryptoStore()
console.log('Has key pair:', cryptoStore.hasKeyPair)
console.log('Public key:', cryptoStore.publicKeyPem)
```

### 查看 WebSocket 状态

```javascript
const networkStore = useNetworkStore()
console.log('Status:', networkStore.connectionStatus)
console.log('Latency:', networkStore.latency)
console.log('Reconnect attempts:', networkStore.reconnectAttempts)
```

### 查看消息队列

```javascript
const messageStore = useMessageStore()
console.log('Pending:', messageStore.pendingCount)
console.log('Messages:', messageStore.messagesMap)
```

## 📝 注意事项

1. **首次登录**：会自动生成 RSA 密钥对，需要几秒钟
2. **密钥存储**：私钥存储在系统安全区域，重启应用后自动加载
3. **WebSocket**：当前连接测试服务器，可能不稳定
4. **Mock 数据**：会话列表使用 Mock 数据，实际需要后端 API
5. **文件拖拽**：UI 已实现，加密上传功能待开发
