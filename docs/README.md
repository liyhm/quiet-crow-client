## 🎯 项目概览

Quiet Crow 是一个基于 Electron + Vue 3 + TypeScript 的端到端加密聊天应用。

### 技术栈

- 前端：Electron + Vue 3 + TypeScript + Pinia + Tailwind CSS
- 后端：Spring Boot + MySQL + Redis
- 加密：AES-ECB 端到端加密
- 通信：WebSocket (SockJS + STOMP)

### 主要功能

- ✅ 用户认证（注册/登录）
- ✅ 端到端加密消息
- ✅ 实时通讯（WebSocket）
- ✅ 会话管理（私聊/群聊）
- ✅ 好友管理
- ✅ 消息历史

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 配置环境
cp .env.example .env.development

# 启动开发服务器
npm run dev
```

## 📁 项目结构

```
src/
├── main/                   # Electron 主进程
├── preload/               # 预加载脚本
└── renderer/              # 渲染进程（Vue 应用）
    └── src/
        ├── app/           # 应用配置（路由、状态管理）
        ├── entities/      # 实体和状态管理（Store）
        ├── pages/         # 页面组件
        ├── shared/        # 共享资源
        │   ├── api/       # API 服务层
        │   ├── config/    # 配置文件
        │   ├── lib/       # 工具库（加密等）
        │   └── types/     # TypeScript 类型定义
        └── widgets/       # 复合组件
```
