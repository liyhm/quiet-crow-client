# 登录页关闭按钮和图标修复

## 修复时间
2026-03-01

## 问题描述

### 问题 1: 登录页无法关闭
- 登录页没有关闭按钮
- 如果登录失败，用户无法关闭应用
- 只能通过任务管理器强制结束进程

### 问题 2: 应用图标不正确
- 打包后的应用图标不是我们的 logo
- 使用的是默认的 Electron 图标

## 解决方案

### 1. 登录页添加关闭按钮 ✅

**位置**: 右上角

**样式**:
- 圆形按钮
- 毛玻璃效果
- Hover 时旋转动画
- 赛博朋克风格

**实现**:

```vue
<!-- 关闭按钮 -->
<button
  @click="closeWindow"
  class="absolute top-4 right-4 w-10 h-10 rounded-full bg-white/5 hover:bg-white/10 border border-white/10 flex items-center justify-center text-white/50 hover:text-white transition-all duration-300 z-50 group"
  title="关闭应用"
>
  <svg
    class="w-5 h-5 transition-transform group-hover:rotate-90"
    fill="none"
    stroke="currentColor"
    viewBox="0 0 24 24"
  >
    <path
      stroke-linecap="round"
      stroke-linejoin="round"
      stroke-width="2"
      d="M6 18L18 6M6 6l12 12"
    />
  </svg>
</button>
```

**功能**:
```typescript
const closeWindow = async () => {
  await window.api.window.close()
}
```

### 2. 应用图标替换指南 ✅

**图标位置**: `resources/icon.png`

**替换步骤**:

1. 准备图标文件
   - 格式: PNG
   - 尺寸: 512x512 或 1024x1024 像素
   - 要求: 正方形，透明背景

2. 替换图标文件
   ```powershell
   # 备份原图标
   Copy-Item resources\icon.png resources\icon.png.backup
   
   # 复制新图标
   Copy-Item 你的图标路径 resources\icon.png
   ```

3. 重新打包
   ```powershell
   Remove-Item -Recurse -Force dist, out
   npm run build:win
   ```

## 注意事项

### 关闭按钮
1. 关闭按钮在所有状态下都可用
2. 即使登录失败也可以关闭应用
3. 关闭按钮不会被其他元素遮挡

### 图标替换
1. 图标必须是正方形
2. 推荐使用透明背景
3. 尺寸至少 512x512 像素
4. 替换后需要重新打包
5. Windows 可能缓存旧图标，需要清理缓存

## 快速参考

### 关闭应用
- 点击右上角的 X 按钮
- 或按 Alt+F4（Windows）
- 或按 Command+Q（macOS）

### 替换图标
```powershell
# 1. 替换图标
Copy-Item 你的图标.png resources\icon.png

# 2. 重新打包
npm run build:win
```
