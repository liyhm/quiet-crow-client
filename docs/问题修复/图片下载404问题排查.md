# 图片下载 404 问题排查

## 📋 问题描述

前端上传图片成功后，尝试下载显示图片时返回 404 错误：

```
GET http://localhost:5173/api/media/78f4f24f8621a797bfa81a4072e97478 404 (Not Found)
```

**错误截图显示**：
- 上传接口返回 200 成功
- 返回了 fileId: `78f4f24f8621a797bfa81a4072e97478`
- 但下载时返回 404

---

## 🔍 问题分析

这是**后端问题**，可能的原因：

1. **文件保存失败**
   - 上传接口返回成功，但文件没有实际保存到磁盘
   - 数据库记录插入成功，但文件保存失败

2. **下载接口路径错误**
   - 后端下载接口的路径映射不正确
   - 前端请求的路径和后端定义的路径不匹配

3. **文件 ID 不存在**
   - 数据库中没有这个 fileId 的记录
   - 或者记录被标记为已删除

4. **权限问题**
   - 后端验证权限时失败，返回 404 而不是 403

## 🛠️ 后端修复方案

### 修复 1: 确保文件保存成功

在上传接口中添加文件保存验证：

```java
@PostMapping("/upload")
public Result<UploadResponse> uploadMedia(
    @RequestParam("file") MultipartFile file,
    @RequestParam("metadata") String metadataJson
) {
    try {
        // ... 前面的代码 ...
        
        // 保存文件到磁盘
        String fileId = IdUtil.fastSimpleUUID();
        String storagePath = buildStoragePath(fileId, metadata.getMessageType());
        
        File destFile = new File(storagePath);
        destFile.getParentFile().mkdirs();
        
        // 保存文件
        file.transferTo(destFile);
        
        // ✅ 验证文件是否保存成功
        if (!destFile.exists() || destFile.length() == 0) {
            throw new RuntimeException("文件保存失败");
        }
        
        log.info("文件保存成功: {}, 大小: {}", storagePath, destFile.length());
        
        // 保存到数据库
        MediaFile mediaFile = new MediaFile();
        mediaFile.setId(fileId);
        mediaFile.setSessionId(metadata.getSessionId());
        mediaFile.setUploaderId(userId);
        mediaFile.setFileName(metadata.getFileName());
        mediaFile.setFileSize(metadata.getFileSize());
        mediaFile.setMimeType(metadata.getMimeType());
        mediaFile.setMessageType(metadata.getMessageType());
        mediaFile.setStoragePath(storagePath);
        
        boolean saved = mediaFileService.save(mediaFile);
        
        // ✅ 验证数据库保存是否成功
        if (!saved) {
            // 删除已保存的文件
            destFile.delete();
            throw new RuntimeException("数据库保存失败");
        }
        
        log.info("数据库保存成功: {}", fileId);
        
        // 返回结果
        UploadResponse response = new UploadResponse();
        response.setFileId(fileId);
        response.setUploadTime(LocalDateTime.now().toString());
        response.setFileSize(metadata.getFileSize());
        response.setThumbnailUrl(null);
        
        return Result.success(response);
        
    } catch (Exception e) {
        log.error("文件上传失败", e);
        return Result.error("文件上传失败: " + e.getMessage());
    }
}
```

### 修复 2: 完善下载接口

```java
@GetMapping("/{fileId}")
public ResponseEntity<Resource> downloadMedia(@PathVariable String fileId) {
    try {
        log.info("下载文件请求: fileId={}", fileId);
        
        // 获取当前用户
        String userId = StpUtil.getLoginIdAsString();
        log.info("当前用户: {}", userId);
        
        // 查询文件信息
        MediaFile mediaFile = mediaFileService.getById(fileId);
        
        // ✅ 检查文件是否存在
        if (mediaFile == null) {
            log.error("文件不存在: {}", fileId);
            return ResponseEntity.notFound().build();
        }
        
        log.info("文件信息: sessionId={}, storagePath={}", 
            mediaFile.getSessionId(), mediaFile.getStoragePath());
        
        // ✅ 检查权限
        if (!sessionService.isMember(mediaFile.getSessionId(), userId)) {
            log.error("无权限访问文件: fileId={}, userId={}", fileId, userId);
            return ResponseEntity.status(HttpStatus.FORBIDDEN).build();
        }
        
        // ✅ 检查文件是否存在于磁盘
        File file = new File(mediaFile.getStoragePath());
        if (!file.exists()) {
            log.error("文件不存在于磁盘: {}", mediaFile.getStoragePath());
            return ResponseEntity.notFound().build();
        }
        
        log.info("文件存在，大小: {}", file.length());
        
        // 读取文件
        Resource resource = new FileSystemResource(file);
        
        // 设置响应头
        HttpHeaders headers = new HttpHeaders();
        headers.add(HttpHeaders.CONTENT_DISPOSITION, "attachment; filename=\"encrypted-file\"");
        headers.add(HttpHeaders.CONTENT_TYPE, MediaType.APPLICATION_OCTET_STREAM_VALUE);
        headers.add("X-File-Name", mediaFile.getFileName());
        headers.add("X-File-Size", String.valueOf(mediaFile.getFileSize()));
        headers.add("X-Mime-Type", mediaFile.getMimeType());
        headers.add("X-Upload-Time", mediaFile.getUploadTime().toString());
        
        log.info("文件下载成功: {}", fileId);
        
        return ResponseEntity.ok()
            .headers(headers)
            .body(resource);
            
    } catch (Exception e) {
        log.error("文件下载失败: fileId={}", fileId, e);
        return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR).build();
    }
}
```
