# 头像上传成功但不显示问题

## 🔍 问题描述

从 Network 面板看到：
1. ✅ 上传请求成功（`/api/media/upload`）
2. ✅ 返回了 `avatarUrl`
3. ✅ 更新用户信息请求成功（`/api/user/info`）
4. ❌ 但是前端头像没有显示

## 📊 可能的原因

### 原因 1: 头像 URL 格式问题

**检查返回的 avatarUrl**:
```json
{
  "avatarUrl": "http://127.0.0.1:9005/img-rtm/image/2026/03/04/494eed06b3c44216b96d87f23ce54de5.jpg"
}
```

**问题**: 前端可能需要完整的 URL，或者需要处理相对路径。

### 原因 2: 用户信息没有及时更新

**当前代码**（MainPage.vue）:
```typescript
const handleAvatarUpload = async (event: Event): Promise<void> => {
  // ... 上传逻辑
  
  try {
    const result = await userApi.uploadAvatar(file)
    console.log('✅ 头像上传成功:', result.avatarUrl)
    
    // 刷新用户信息
    await authStore.fetchUserInfo()
    console.log('✅ 用户信息已刷新')
    
    showAlert('头像上传成功！', 'success')
  } catch (error) {
    // ...
  }
}
```

**可能的问题**:
- `authStore.fetchUserInfo()` 可能没有正确更新 `currentUser`
- ProfileModal 使用的 `user` prop 可能没有响应式更新

### 原因 3: 图片加载失败（CORS 或路径问题）

**检查**:
1. 打开浏览器开发者工具 Console
2. 查看是否有图片加载错误
3. 检查 Network 面板中的图片请求

### 完整的修复代码

**MainPage.vue**:
```typescript
const handleAvatarUpload = async (event: Event): Promise<void> => {
  const target = event.target as HTMLInputElement
  const file = target.files?.[0]

  if (!file) {
    console.log('ℹ️ 没有选择文件')
    return
  }

  console.log('📤 开始上传头像:', file.name, (file.size / 1024).toFixed(2), 'KB')

  if (!file.type.startsWith('image/')) {
    showAlert('请选择图片文件', 'error')
    return
  }

  if (file.size > 5 * 1024 * 1024) {
    showAlert('图片大小不能超过5MB', 'error')
    return
  }

  uploadingAvatar.value = true

  try {
    console.log('🔄 调用上传 API...')
    const result = await userApi.uploadAvatar(file)
    console.log('✅ 头像上传成功:', result.avatarUrl)
    
    console.log('🔄 刷新用户信息...')
    await authStore.fetchUserInfo()
    console.log('✅ 用户信息已刷新')
    console.log('📸 当前用户头像:', authStore.currentUser?.avatar)
    
    // 强制刷新弹窗
    showProfileModal.value = false
    await nextTick()
    showProfileModal.value = true
    
    showAlert('头像上传成功！', 'success')
  } catch (error: any) {
    console.error('❌ 头像上传失败:', error)
    
    let errorMessage = '未知错误'
    if (error.message) {
      errorMessage = error.message
    } else if (error.response?.data?.message) {
      errorMessage = error.response.data.message
    } else if (error.code === 'ECONNABORTED') {
      errorMessage = '上传超时，请检查网络连接'
    }
    
    showAlert('头像上传失败: ' + errorMessage, 'error')
  } finally {
    uploadingAvatar.value = false
    target.value = ''
    console.log('🏁 头像上传流程结束')
  }
}
```
