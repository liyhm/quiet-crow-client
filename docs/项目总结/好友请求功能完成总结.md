## 📊 数据流

### 收到新的好友请求

```
用户 B 发送请求
    ↓
后端推送 WebSocket 通知
    ↓
前端接收通知 (NEW_REQUEST)
    ↓
调用 contactStore.loadPendingRequests()
    ↓
更新 pendingRequests 数组
    ↓
computed 属性自动计算 pendingRequestCount
    ↓
侧边栏红色数字自动更新
    ↓
显示桌面通知
```

### 好友请求被接受

```
用户 B 接受请求
    ↓
后端推送 WebSocket 通知
    ↓
前端接收通知 (REQUEST_ACCEPTED)
    ↓
调用 contactStore.loadPendingRequests()
调用 contactStore.loadContacts()
    ↓
更新 pendingRequests 和 contacts 数组
    ↓
computed 属性自动计算
    ↓
侧边栏红色数字自动更新
联系人列表自动显示新好友
    ↓
显示桌面通知
```

private showDesktopNotification(title: string, body: string) {

    new Notification(title, { body })
  } else if ('Notification' in window && Notification.permission !== 'denied') {
    Notification.requestPermission().then(permission => {
      if (permission === 'granted') {
        new Notification(title, { body })
      }
    })
  }
}
