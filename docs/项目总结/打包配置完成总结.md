## 常见问题

### 1. 打包失败

**原因**: 依赖缺失或网络问题

**解决**:
```bash
npm cache clean --force
rm -rf node_modules
npm install
npm run build:win
```

### 2. 打包后无法连接后端

**原因**: 生产环境配置错误

**解决**: 检查 `.env.production` 文件，确保地址正确

### 3. 安装包太大

**原因**: 包含了不必要的文件

**解决**: 在 `electron-builder.yml` 中添加排除规则

### 4. 打包速度慢

**原因**: 网络下载 Electron 二进制文件慢

**解决**: 已配置国内镜像，如果还是慢可以使用代理

## 后续优化

### 1. 使用 HTTPS

建议后端使用 HTTPS，提高安全性：
```env
VITE_API_BASE_URL=https://api.your-domain.com
VITE_WS_URL=https://api.your-domain.com/ws
```

### 2. 使用域名

使用域名代替 IP 地址：
```env
VITE_API_BASE_URL=https://api.quiet-crow.com
VITE_WS_URL=https://api.quiet-crow.com/ws
```

### 3. 配置自动更新

在 `electron-builder.yml` 中配置：
```yaml
publish:
  provider: generic
  url: https://your-domain.com/updates
```

### 4. 代码签名

Windows 和 macOS 应用签名，避免安全警告：
- Windows: 需要代码签名证书
- macOS: 需要 Apple Developer 账号

### 5. 减小体积

优化打包配置，减小安装包体积：
```yaml
compression: maximum
asar: true
```
