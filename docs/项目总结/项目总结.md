## 🎯 核心功能演示流程

### 1. 启动与登录

```
npm run dev
    ↓
显示登录页（微信绿色渐变背景）
    ↓
输入用户名/密码
    ↓
点击登录
    ↓
生成 RSA 密钥对（2048-bit）
    ↓
私钥存入 OS 安全存储
    ↓
初始化 WebSocket 连接
    ↓
跳转主界面
```

### 2. 发送加密消息

```
输入消息内容
    ↓
按 Enter 键
    ↓
获取会话 AES 密钥
    ↓
AES-GCM 加密（生成 IV + AuthTag）
    ↓
通过 WebSocket 发送
    ↓
消息状态：发送中 → 已发送
    ↓
界面显示绿色气泡
```

### 3. 接收解密消息

```
WebSocket 接收消息
    ↓
提取密文 + IV + AuthTag
    ↓
获取会话 AES 密钥
    ↓
AES-GCM 解密
    ↓
添加到消息列表
    ↓
更新会话最新消息
    ↓
增加未读数（如果非当前会话）
    ↓
界面显示白色气泡
```

## 🔐 安全特性

### 端到端加密

- ✅ 消息内容使用 AES-GCM 256-bit 加密
- ✅ 每条消息独立 IV（12 字节随机）
- ✅ 认证标签防篡改（128-bit）
- ✅ 服务器无法解密消息内容

### 密钥保护

- ✅ RSA 私钥存储在 OS 安全区域
  - Windows: DPAPI 加密
  - macOS: Keychain 加密
- ✅ 即使文件被盗，无系统密码无法解密
- ✅ AES 会话密钥仅存在内存中

### 通信安全

- ✅ WebSocket 支持 TLS（wss://）
- ✅ JWT Token 认证
- ✅ 心跳检测防连接劫持
