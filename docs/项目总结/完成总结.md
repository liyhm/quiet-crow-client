## 📝 使用说明

### 开发环境

1. **配置开发环境**

   ```bash
   # .env.development 已配置为 localhost:8080
   # 无需修改，直接使用
   ```

2. **启动开发服务器**

   ```bash
   npm run dev
   ```

3. **测试 API**
   - 参考 [API测试指南.md](./API测试指南.md)

### 生产环境

1. **配置生产环境**

   ```bash
   # 编辑 .env.production
   vim .env.production
   ```

   修改为实际的生产服务器地址：

   ```env
   VITE_API_BASE_URL=https://api.your-domain.com
   VITE_WS_URL=wss://api.your-domain.com/ws
   ```

2. **构建应用**

   ```bash
   npm run build:win   # Windows
   npm run build:mac   # macOS
   npm run build:linux # Linux
   ```

3. **部署检查**
   - 参考 [部署检查清单.md](./部署检查清单.md)

## 🔍 验证方法

### 验证环境配置

```bash
# 开发环境
npm run dev
# 打开浏览器控制台
console.log(import.meta.env.VITE_API_BASE_URL)
# 应该输出: http://localhost:8080
```

### 验证 API 调用

```javascript
// 在浏览器控制台
const { authApi } = await import('./src/renderer/src/shared/api/auth')
const result = await authApi.login({
  username: 'testuser',
  password: '123456'
})
console.log(result)
```

### 验证加密功能

```javascript
// 在浏览器控制台
const { backendCrypto } = await import('./src/renderer/src/shared/lib/cryptoBackend')
const testKey = 'xK9mP2vN8qR5tY7wZ3bC6dF1gH4jL0sA=='
backendCrypto.setSessionKey('test', testKey)
const encrypted = backendCrypto.encryptWithSession('test', 'Hello')
const decrypted = backendCrypto.decryptWithSession('test', encrypted)
console.log(decrypted) // 应该输出: "Hello"
```

## ⚠️ 重要提示

### 部署前必做

1. **修改生产环境配置**
   - 编辑 `.env.production`
   - 替换为实际的生产服务器地址
   - 确保使用 HTTPS/WSS

2. **后端服务器准备**
   - SSL 证书已配置
   - WebSocket 已启用
   - CORS 已正确配置
   - 防火墙规则已设置

3. **测试验证**
   - 测试登录功能
   - 测试消息发送
   - 测试 WebSocket 连接
   - 检查网络请求地址

### 安全建议

1. **不要提交敏感信息**
   - `.env.production` 可能包含生产地址
   - 建议添加到 `.gitignore`（已配置）

2. **使用 HTTPS/WSS**
   - 生产环境必须使用加密连接
   - 确保 SSL 证书有效

3. **Token 管理**
   - Token 存储在 localStorage
   - 定期刷新 Token
   - Token 过期自动跳转登录
