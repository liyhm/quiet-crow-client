## 🔧 技术栈

- **桌面框架**: Electron 39.x
- **前端框架**: Vue 3.5 (Composition API)
- **构建工具**: Vite 7.x + electron-vite 5.x
- **语言**: TypeScript 5.x
- **状态管理**: Pinia 3.x
- **路由**: Vue Router 5.x
- **样式**: Tailwind CSS 4.x
- **通信**: @stomp/stompjs + SockJS
- **加密**: Web Crypto API
- **测试**: Vitest + @vue/test-utils

## 📝 开发命令

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 类型检查
npm run typecheck

# 代码格式化
npm run format

# 代码检查
npm run lint

# 构建
npm run build

# 打包 Windows
npm run build:win

# 打包 macOS
npm run build:mac

# 打包 Linux
npm run build:linux
```
