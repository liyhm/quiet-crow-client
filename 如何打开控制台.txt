===============================================
  如何打开控制台（开发者工具）
===============================================

方式 1: 3 次连击 Ctrl 键（推荐）⭐
  - 快速按 3 次 Ctrl 键（0.8 秒内）
  - Windows/Linux: Ctrl 键
  - macOS: Command 键
  - 隐蔽，不容易被误触

方式 2: F12 快捷键
  - 按 F12 键
  - 标准的开发者工具快捷键

方式 3: Ctrl+Shift+I
  - Windows/Linux: Ctrl+Shift+I
  - macOS: Command+Shift+I
  - Chrome 浏览器标准快捷键

方式 4: Ctrl+Shift+C
  - Windows/Linux: Ctrl+Shift+C
  - macOS: Command+Shift+C
  - 打开元素选择器

===============================================
  使用场景
===============================================

1. 查看 API 地址配置
   - 打开控制台
   - 查找 "🔧 API 配置" 日志

2. 查看登录状态
   - 打开控制台
   - 查找 "✅ 登录成功" 日志

3. 查看 WebSocket 连接
   - 打开控制台
   - 查找 "🔌 WebSocket" 日志

4. 查看错误信息
   - 打开控制台
   - 查找红色的 "❌" 错误日志

===============================================
  注意事项
===============================================

1. 3 次连击必须在 0.8 秒内完成
2. 确保应用窗口有焦点
3. 再次按快捷键可以关闭控制台
